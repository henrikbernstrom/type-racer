# Server

## Prerequisites

- Node.js 18+
- npm 9+

## Install

```sh
npm install
```

If installing from the repository root with workspaces:

```sh
npm install
```

## Run (development)

```sh
npm run dev
```

From the repository root (using workspaces):

```sh
npm run dev -w server
```

## Build

```sh
npm run build
```

From the repository root:

```sh
npm run build -w server
```

## Start (after build)

```sh
npm run start
```

From the repository root:

```sh
npm run start -w server
```

## Test

```sh
npm run test
```

From the repository root:

```sh
npm run test -w server
```
